(1) 프로젝트 명칭
경량화된 팀 협업을 위한 분산 파일 관리 및 히스토리 추적 시스템 구축 

(2) 개발 시스템 명칭
에어 드라이브 (Air Drive) 

(3) 개발 시스템 설명
본 시스템은 대규모 클라우드 서비스의 복잡함을 걷어내고, 팀 단위의 빠른 자료 공유에 최적화된 경량 웹 스토리지로, 
사용자는 웹을 통해 파일을 업로드하고, 팀원 간의 실시간 댓글 피드백을 통해 문서의 수정 방향을 공유할 수 있음
(저장 공간 효율을 위해 변경된 부분만 관리하는 버전 제어 기능을 제공하여 팀의 데이터 자산을 안전하게 보호할 수 있음)

(4) 개발 시스템 핵심 기능 (구현 가능 중심)
드래그 앤 드롭 기반 파일 관리: 복잡한 설치 없이 브라우저에서 파일을 바로 올리고 폴더별로 분류하는 직관적 관리 기능. 
시간 제한부 공유 링크 생성: 보안을 위해 특정 시간(예: 1시간, 24시간) 동안만 활성화되는 임시 다운로드 링크 발급 기능. 
파일별 협업 댓글: 파일 자체에 댓글을 달아 수정 이력이나 팀원 간의 의견을 기록하고 보존하는 기능. 
최근 활동 로그 추적: 누가 어떤 파일을 언제 수정하거나 다운로드했는지 실시간으로 보여주는 가시성 제공 기능. 
키워드 기반 고속 필터링: 파일명과 확장자, 태그를 조합하여 수천 개의 파일 중 필요한 자료를 즉시 찾아내는 검색 기능. 

(5) 기존 유사 소프트웨어 조사 및 분석

파일 전송 전문 서비스 (예: SendAnywhere): 1회성 전송은 빠르지만, 파일의 버전 관리나 팀 단위의 지속적인 보관 및 협업이 불가능함. 
엔터프라이즈 솔루션 (예: MS OneDrive): 기능은 강력하나 설정이 매우 복잡하고 시스템 리소스 소모가 커 소규모 팀이 가볍게 쓰기엔 진입 장벽이 높음.
